# Julia P Braids

Portfólio profissional para uma trancista especializada em tranças nagô, com foco em apresentação de serviços, atração de clientes locais e conversão pelo WhatsApp.

## Visão Geral

O site posiciona a Julia P Braids como uma opção elegante e personalizada para quem busca tranças nagô em Itapevi e região.

A página destaca:

- Tranças nagô tradicionais
- Tranças desenhadas
- Freestyle
- Detalhes e acessórios
- Atendimento em local próprio ou a domicílio
- Orçamento personalizado pelo WhatsApp
- Pagamentos combinados diretamente com a cliente, sem necessidade de maquininha

## Estratégia Comercial

O serviço não trabalha com preços fixos. O orçamento é definido em conversa direta com a cliente, considerando:

- Tipo de trança
- Complexidade do modelo
- Tempo de execução
- Materiais necessários
- Local de atendimento
- Deslocamento, quando houver

O fluxo principal do site leva a cliente para o WhatsApp, onde ela pode enviar referências, informar sua região e combinar o atendimento.

## Estrutura

- `index.html`: estrutura da página e conteúdo comercial
- `style.css`: identidade visual, responsividade e acabamento
- `script.js`: animações, menu mobile e carrossel
- `static/images/`: logos, imagens otimizadas e ativos do site
- `static/images/web/`: imagens leves usadas na página publicada
- `vercel.json`: configuração simples para publicação na Vercel
- `.vercelignore`: arquivos que não devem ir para o deploy

## Publicação na Vercel

Este é um projeto estático. Para publicar:

1. Suba o repositório para o GitHub.
2. Importe o projeto na Vercel.
3. Mantenha o framework como `Other`.
4. Não configure comando de build.
5. Use `index.html` como entrada principal.

As imagens leves usadas pelo site estão em `static/images/web/`. Os ativos e logos ficam em `static/images/`.
